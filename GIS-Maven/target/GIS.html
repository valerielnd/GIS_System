  User Tutorial

• To Sign up for GIS and create a new account first click the “Sign Up” Button on the Sign in page.
 You will be directed to the sign up pop up window.

• Enter your email as well as your unique security question and Password. (Never send or share your password to others)

• If your login email is valid and new then your new account will be created and you will be redirected to the login page to sign in

• To sign into the GIS interface, open the app and fill in your email and password. Then click the “Sign In” Button.

• If your log in information is valid you will be directed to the Campus Map Screen.

• If you are encountering errors please consult the reset password section of the help site.

• To reset you password in the case of it being forgotten or compromised, first click the “Forgot Password” Button on the sign in page. You will be directed to the forgotten password Pop-Up

• Input the email you previously used to sign up, after entering your email you will be prompted to enter the security question you chose when setting up your account

• After entering your security Question you will now be able to enter and confirm your new password to log in with

• At the left tool box, select the drop box menu for building

• This will only show the buildings that weres added.

• At the left tool box, select the drop box menu for building

• The floor drop box is avaiable afer the building is selected

• POIs drop box will be show and can be select base on POIs name

• POIs will only show the POIs that weres added.

•To add a Custom POI to the map, first navigate to the floor you want to add the POI to

•Click on the Add POI to bring you to the POI information pop up.

•Fill in the information for the POI name and description. You can also chose to add the POI to your list of favourites

•Click “Save” to store the POI and add it to your Custom POI list

• Select one POI in the favourite box from the left tool box

• Select edit POI drop box and choose edit

• Edit POI option: change name ; change description , set the POI into favourite

• To Remove a Custom POI use the left side “Custom POI” Menu to select the Custom POI you want to delete

• Right Click the custom POI to bring up the edit button

• Left click the edit button to expand the edit drop down and click on “Delete”

• You will be directed to the delete POI popup

• If you click delete you will get a confirmation Pop-up then confirming will delete the POI

• In the bottom left hand side, there is a Layer box

• Select a layer listed in the layer box to turn in on or off


  Developer Tutorial

• Select the location of where you want to place the building on the campus map

• Click on the "Edit Building" button on the top left hand side of the window

• In the drop down list, select "+ Add Building" button

• Add the building information you want in the new popup

• Click "save"

• On the top left hand side, click on "Edit building"

• Select the building you want to edit in the drop down menu

• Fill in the information for the building in the new popup

• Select "Save"

• On the top left hand side, select "Edit Buildings"

• Select a building in the drop down menu that just appeared

• On the new popup, click "Delete"

• On the left side, above the Built-in POIs box, there is a "Add" button. Select it

• Fill in the inforamtion in the new popup window

• Select "Save" once done

• In the built-in POIs box, right click on a POI in that box

• Beside the "Add" button is now a new "Edit" button. Select it

• In the new drop down, click edit

• In the popup, fill in the information you want. Then select "save"

• In the Built-in POIs box, right click a POI

• A new "Edit" button will appear above the box. Select it

• In the drop down, select delete

• In the top left hand side, select "Edit Building"

• In the drop down, select the building of the floor you want to add

• Select add floor from the new popup

• Add the floor information to the new popup and then hit "Save"

• In the top left hand side, select "Edit Building"

• Select the building for the floor you want to edit in the drop down menu

• Select edit floor in the new popup

• Select the floor number in the drop down menu

• Edit in the information in the new popup and then select "Save"

• In the top left hand side, select "Edit Building"

• Select the building for the floor you want to delete in the drop down menu

• In the new popup, click "Delete"
